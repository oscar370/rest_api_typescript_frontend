# Uptask

Frontend for a school project on a Trello-style task and project manager

Backend: [https://github.com/oscar370/rest_apis_typescript_backend](https://github.com/oscar370/rest_apis_typescript_backend)

## Tech Stack

### Frontend

- React
- Typescript
- React Router DOM Actions

### Backend

- Postgres
- Sequelize
- Express

## Development

### Prerequisites

- [Node.js](https://nodejs.org/)
- [pnpm](https://pnpm.io/)

### Installation

1. Clone the repository:

```bash
git clone https://github.com/oscar370/rest_api_typescript_frontend
cd rest_api_typescript_frontend
```

2. Install dependencies:

```bash
pnpm install
```

3. Start the development server:

```bash
pnpm dev
```

4. Build for production:

```bash
pnpm build
```
